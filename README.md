# Gerador de Senhas 🔐

Projeto simples que gera senhas aleatórias.

## 🚀 Tecnologias
- HTML
- CSS
- JavaScript

## 📌 Funcionalidades
- Gerar senha aleatória
- Escolher tamanho da senha
- Copiar senha ao clicar

## ▶️ Como usar
Abra o arquivo `index.html` no navegador.
